# AIMS

My first year at UWEC I joined UWEC's Artificial Intelligence and Machine Learning Security research group (AIMS) as a student researcher.

## 2023-2024

My first year I started work on using machine learning to analyze network logs and detect malicious traffic. After analyzing a number of network traffic datasets, we settled on using the [NF-ToN-IoT-v2](https://doi.org/10.48610/38a2d07) dataset. We found that using ML algorithms that the authros used in their orginal study we could get a preditction accuracy < 98%, which was very impressive. Our next steps are using algorithms that the authors haven't used with the dataset, such as K Nearest Neighbor and Support Vector Machine, maybe even a Convolution Neural Network. Research is ongoing. 

[Here](/static/AIMSPoster.pdf) is the poster we presented at [Cerca](https://www.uwec.edu/orsp/students/cerca/)
